Thank you for installing MDnote!

WHAT IS MDNOTE?
---------------
A fast, lightweight Markdown viewer for Windows with:
- Dark & Light themes
- Font size adjustment
- Recent files history
- Zero external dependencies


HOW TO USE
----------
1. Double-click any .md file to open it
2. Or find "MDnote" in your Start Menu


KEYBOARD SHORTCUTS
------------------
Ctrl + O     Open file
Ctrl + T     Toggle theme (Dark/Light)
Ctrl + +     Increase font size
Ctrl + -     Decrease font size
Ctrl + 0     Reset font size


FEATURES
--------
[x] Dark & Light themes
[x] Font scaling
[x] Recent files (last 5)
[x] Status bar (lines/words/chars)
[x] Always on top option
[x] Extended Markdown syntax
    - Headers (H1-H6)
    - Code blocks & inline code
    - Bold, italic, strikethrough
    - Lists & checkboxes
    - Horizontal rules


SETTINGS LOCATION
-----------------
Your settings are saved at:
%APPDATA%\MDnote\config.json

Recent files:
%APPDATA%\MDnote\recent.json


UNINSTALL
---------
Run the uninstaller from:
- Control Panel > Programs
- Or Start Menu > MDnote > Uninstall
